Initial release of @kubb/plugin-pinia-colada

- Generate type-safe Pinia Colada hooks from OpenAPI specifications
- Support for `useQuery` hooks for GET requests
- Support for `useMutation` hooks for POST, PUT, PATCH, DELETE requests
- Full TypeScript support with type inference
- Compatible with Pinia Colada's API (`@pinia/colada`)
- Supports Zod validation integration
- Configurable query and mutation keys
- Works seamlessly with @kubb/plugin-client for HTTP client generation
