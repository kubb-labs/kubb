Client used for HTTP calls.

|           |                      |
|----------:|:---------------------|
|     Type: | `'axios' \| 'fetch'` |
| Required: | `false`              |
|  Default: | `'axios'`            |


- `'axios'` uses `@kubb/plugin-client/templates/axios` to fetch data.
- `'fetch'` uses `@kubb/plugin-client/templates/fetch` to fetch data.
