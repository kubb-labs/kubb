Which client should be used to do the HTTP calls

|           |                      |
|----------:|:---------------------|
|     Type: | `'axios' \| 'fetch'` |
| Required: | `false`              |
|  Default: | `'axios'`            |


- `'axios'` will use `@kubb/plugin-client/clients/axios` to fetch data.
- `'fetch'` will use `@kubb/plugin-client/clients/fetch` to fetch data.
