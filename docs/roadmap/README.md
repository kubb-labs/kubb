# Kubb Roadmap

This directory contains planning documents for future Kubb features and improvements.

## Documents

### [OpenAPI Feature Gaps](./openapi-feature-gaps.md)

Comprehensive analysis of missing OpenAPI v3.0 and v3.1 features with implementation roadmap.

**Key Missing Features:**
- ❌ Webhooks (OpenAPI 3.1)
- ❌ Callbacks (OpenAPI 3.0)
- ❌ Links (OpenAPI 3.0)
- ❌ if/then/else conditional schemas (JSON Schema 2020-12)
- ❌ $defs (JSON Schema 2020-12)
- ❌ Advanced JSON Schema 2020-12 features

## Contributing

If you'd like to contribute to implementing any of these features:

1. Check the feature gaps document for implementation details
2. Create or comment on relevant GitHub issues
3. Follow the contribution guidelines in [CONTRIBUTING.md](../../CONTRIBUTING.md)
4. Discuss your approach with maintainers before starting large features

## Feedback

Have suggestions for the roadmap? Please:
- Open a GitHub issue with the `enhancement` label
- Discuss in GitHub Discussions
- Reach out to maintainers
