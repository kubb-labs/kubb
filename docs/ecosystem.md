# Kubb Ecosystem

**Discover how developers and teams use Kubb** in production applications, articles, and open-source projects. This page showcases the growing Kubb ecosystem.

## Production Projects Using Kubb

Real-world applications and frameworks powered by Kubb's OpenAPI code generation:

- **[KOPER](https://github.com/dxloop/koper)** - Modern full-stack application framework powered by Kubb for type-safe API integration
- **[Xata MCP Server](https://xata.io/blog/built-xata-mcp-server)** - Xata's Model Context Protocol server built using Kubb's OpenAPI to MCP generation
- **[ClickHouse + MooseStack](https://clickhouse.com/blog/clickhouse-powered-apis-in-react-app-moosestack#bridging-to-the-frontend)** - ClickHouse-powered APIs in React apps using Kubb
- **[Fiveonefour](https://docs.fiveonefour.com/moose/apis/openapi-sdk)** - OpenAPI SDK generation powered by Kubb

## Resources for Teams

### Enterprise Use Cases
Kubb is production-ready and used by teams for:
- **Microservices** - Generate clients for multiple internal APIs
- **Frontend/Backend sync** - Keep TypeScript types synchronized with backend changes
- **Monorepos** - Share generated types across multiple frontend applications
- **CI/CD integration** - Automate code generation in build pipelines
