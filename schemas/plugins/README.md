# Plugin Registry Schema

## `plugins.json`

Schema for the aggregated plugin registry maintained in kubb.dev. This lists all known plugins and is consumed by the docs site to render the plugin directory.

**Schema URL**: `https://kubb.dev/schemas/plugins.json`

---

For the per-package extension manifest schema, see [`../README.md`](../README.md) and [`../extension.json`](../extension.json).
