# Parser Registry Schema

## `parsers.json`

Schema for the aggregated parser registry maintained in kubb.dev. This lists all known parsers and is consumed by the docs site to render the parser directory.

**Schema URL**: `https://kubb.dev/schemas/parsers.json`
