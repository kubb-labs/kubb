# Adapter Registry Schema

## `adapters.json`

Schema for the aggregated adapter registry maintained in kubb.dev. This lists all known adapters and is consumed by the docs site to render the adapter directory.

**Schema URL**: `https://kubb.dev/schemas/adapters.json`

---

For the per-package extension manifest schema, see [`../README.md`](../README.md) and [`../extension.json`](../extension.json).
